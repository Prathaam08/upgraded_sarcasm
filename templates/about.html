About This Project

This Sarcasm Detection web app is built using Python, Flask, and machine learning techniques to detect sarcasm in text, considering contextual information like the situation and the person you're talking to.

It aims to make text analysis more accurate by incorporating real-world conversational context.

You can check the sarcasm in any sentence, view your history, and provide feedback to help improve the model.
